## Nullstellensatz von Bolzano
$\text{Ist } a < b \text{ , } f: [a,b] \to \mathbb{R} \text{ stetig, } f(a) > 0 \land f(b) < 0 \text{, so hat } f \text{ mind. eine NS in } [a,b]$
